# Release 1.0

## Цель

Первый преподаватель должен суметь провести через платформу реальную группу учеников.

## Пользовательский путь

1. Преподаватель регистрируется.
2. Добавляет учеников.
3. Создаёт домашнее задание.
4. Ученики получают его.
5. Ученик отправляет решение.
6. AI проверяет решение.
7. Learning DNA обновляется.
8. Родитель получает отчёт.
9. Преподаватель видит изменения.

## Критерий готовности

Хотя бы одна реальная группа может пользоваться системой без ручного ведения Excel.
