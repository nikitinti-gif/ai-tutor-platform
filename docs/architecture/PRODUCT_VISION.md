# PRODUCT VISION

## Миссия

Создать AI-платформу, которая усиливает преподавателя, а не заменяет его.

## Для кого

- Преподаватели ЕГЭ/ОГЭ
- Онлайн-школы
- Репетиторы
- Родители

## Главная ценность

Не проверка домашних заданий.

Не генерация задач.

А построение индивидуальной траектории обучения на основе навыков.

## Принципы

- Skill First
- Explainable AI
- Human in the Loop
- Learning DNA
- Evidence Based Education

## Что НЕ делает система

- Не выставляет окончательные оценки вместо преподавателя.
- Не принимает необъяснимые решения.
- Не заменяет учителя.
