# Learning DNA

Learning DNA — это цифровая модель обучения ученика.

Цель:
не просто хранить оценки, а понимать, как ученик учится.

Learning DNA отвечает на вопрос:

> Что система знает об ученике и какое следующее действие будет для него полезным?

## Основные блоки

1. Identity — кто ученик.
2. Knowledge — какие темы изучает.
3. Skills — какие навыки развиты.
4. Learning Signals — учебные сигналы.
5. Memory — история взаимодействий.
6. Behaviour — учебное поведение.
7. Motivation — мотивация и вовлечённость.
8. Trajectory — индивидуальная траектория.
9. Predictions — прогноз результата.
10. AI Notes — заметки ИИ для преподавателя.

## Главный принцип

Каждое действие ученика должно обновлять Learning DNA.

Пример:

Ученик отправил решение → AI проверил → найдена ошибка → Learning Signal обновлён → траектория изменилась.
